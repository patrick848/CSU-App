Password Reset


IS&T has introduced Self-Service Password Resets for CampusNet passwords.

Your CSU ID and password are used to access CampusNet, General Computer Labs, the University network, your email account,

employee self-service, the Blackboard elearning system, ePortfolio and several other systems.


The next time you log on to CampusNet, you will be prompted to enter a text enabled cell phone number and/or an alternate non-CSU email address.

These will be used in the future to send you an authorization code that will allow you to reset your CampusNet password without contacting the IS&T Call Center.

(If you choose not to enter a cell phone number or an alternate email address, you will be prompted to do so in another 6 months.)


If you need to reset your CampusNet password, you can go to the CampusNet login page and select "Forgot your Password".

This will allow you to start a process that will email or text an authorization code to you that can be used to change your password.

The authorization code is only valid for 30 minutes.


You should enter the authorization code when prompted and this will allow you to set your password to the value you want it to be.

CampusNet is available 24 hrs/day with the exception of Saturday from 6PM until Sunday at 10AM.


Important: To use this feature, you must logon to CampusNet and enter a text enabled cell phone number and/or an alternate non-CSU email address.


You can reset you password in three easy steps:


1. Logon to CampusNet enter a text enabled cell phone number and/or an alternate non-CSU email address.
2. If you need to reset your password: Go to CampusNet, login and select "Forgot your Password".
3. You will receive an email or text with an authorization code that can be used to change your password.


If for some reason the process does not work for you, you should contact the IS&T Call Center at (216) 687-5050 during normal support hours to have your password reset.
